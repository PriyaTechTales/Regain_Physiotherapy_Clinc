# Regain_Physiotherapy_Clinc

## Regain Physiotherapy Clinic & Home Health Care – Landing Page

This repository contains a single‑page responsive website for *Regain Physiotherapy Clinic & Home Health Care*.  
It is built with plain *HTML, CSS and vanilla JavaScript*, with no external build tools.

### Features

- *Modern hero section* with background illustration, call‑to‑action buttons, and smooth scroll.
- *About section* highlighting the doctor, credentials, and years of experience.
- *Services grid* for therapies, modalities, and consultation types.
- *Credentials / Certificates gallery* with responsive images.
- *Clinic & therapy gallery* with hover effects.
- *Testimonials slider* implemented using horizontal scroll.
- *Appointment form* wired to FormSubmit for email delivery.
- *FAQ accordion* for common questions.
- *Sticky navbar* with mobile hamburger menu.
- *Scroll‑to‑top button* and subtle scroll‑in animations.
- Fully responsive layout for *desktop, tablet, and mobile*.

### Project structure

- index.html – main page markup.
- style.css – all styles, theme colors, responsive layout and animations.
- script.js – JavaScript for:
  - mobile navigation toggle,
  - scroll‑reveal animations,
  - FAQ accordion behaviour,
  - scroll‑to‑top button.
- Photos/ – clinic images, hero background, certificates, and service photos.

### How to run locally

1. *Clone or download* this folder onto your machine.
2. Open the folder in your editor (e.g. VS Code / Cursor).
3. Use one of the following:
   - Double‑click index.html to open it in your browser, or
   - Use an extension like *Live Server* and “Open with Live Server” on index.html.

No additional dependencies or build steps are required.

### Deployment notes

- Any static hosting service works (e.g. GitHub Pages, Netlify, Vercel, or a simple shared hosting account).
- Ensure the Photos/ folder is uploaded in the *same relative path* as in development so image URLs continue to work.

### Customization

- *Clinic details* (name, address, phone, email, Instagram link) can be edited directly in index.html under the footer and hero sections.
- *Colors & fonts* can be changed in style.css at the top under the :root CSS variables.
- *Services list & FAQs* are in index.html — edit or reorder the relevant <section> blocks.
